### Prerequisites

* [.NET 6.0 SDK](https://dot.net/get-dotnet6)


### Run

To run the console application:

```
dotnet run --project AoC.CLI
```

To run the Blazor WebAssembly application:

```
dotnet run --project AoC.WasmUI
```


### Test

```
dotnet test
```
